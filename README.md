# ImageKit - 图片处理神器

> 完全免费 · 本地处理 · 隐私安全 · 比 TinyPNG 更强大

---

## 📖 项目简介

ImageKit 是一款基于浏览器本地处理的在线图片工具，提供图片压缩、格式转换、尺寸调整、裁剪、旋转等功能。

### ✨ 核心特点

- ✅ **完全免费**：所有功能永久免费，无隐藏收费
- ✅ **本地处理**：图片在浏览器本地处理，不上传服务器
- ✅ **隐私安全**：您的图片永远不会离开您的设备
- ✅ **功能强大**：比 TinyPNG 更全能的图片处理能力
- ✅ **现代化UI**：简洁美观，操作直观
- ✅ **批量处理**：一次处理多达 100 张图片

---

## 🎯 产品定位

为设计师、自媒体创作者、电商卖家等用户提供专业的图片处理工具，解决日常图片处理需求。

## 🚀 核心功能

### MVP 功能（第1版）

1. **图片压缩** ⭐⭐⭐⭐⭐
   - 智能压缩，平衡质量和大小
   - 自定义质量（0-100%）
   - 实时预览对比
   - 显示压缩率

2. **格式转换** ⭐⭐⭐⭐⭐
   - 支持 JPG ↔ PNG ↔ WebP ↔ BMP
   - 批量转换
   - 保持原始质量选项

3. **批量处理** ⭐⭐⭐⭐⭐
   - 拖拽上传多张图片
   - 统一操作
   - 批量下载（ZIP打包）

4. **尺寸调整** ⭐⭐⭐⭐
   - 按像素/百分比调整
   - 保持比例
   - 常用预设

5. **图片裁剪** ⭐⭐⭐⭐
   - 可视化裁剪框
   - 固定比例裁剪
   - 自由裁剪

6. **旋转/翻转** ⭐⭐⭐
   - 90°/180°/270° 旋转
   - 水平/垂直翻转

### V2 功能（后期迭代）

- 添加水印（文字/图片）
- 图片拼接（横向/纵向/九宫格）
- AI 背景移除
- 更多滤镜效果

## 🛠 技术栈

### 前端框架
- **React 18** - UI 框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具

### UI 组件
- **shadcn/ui** - 现代化组件库
- **TailwindCSS** - CSS 框架
- **Lucide React** - 图标库

### 核心库
- **browser-image-compression** - 图片压缩
- **Fabric.js** - 图片处理（裁剪、滤镜等）
- **react-dropzone** - 拖拽上传
- **JSZip** - 批量下载打包

### 状态管理
- **Zustand** - 轻量级状态管理

### 部署
- **Vercel** - 静态托管，免费
