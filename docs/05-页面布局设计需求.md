# ImageKit 页面布局设计需求（AI设计专用文档）

> **本文档用途**：提供给 AI 设计工具（如 Figma AI、Midjourney、v0.dev 等）生成页面设计稿

---

## 1. 设计风格与原则

### 1.1 整体风格
- **设计语言**：现代简约、扁平化
- **参考风格**：TinyPNG + Linear.app + Vercel
- **色彩基调**：干净、专业、科技感
- **用户体验**：直观、高效、无学习成本

### 1.2 设计原则
1. **极简主义**：去除一切不必要的元素
2. **功能优先**：UI 服务于功能，不喧宾夺主
3. **一致性**：统一的间距、圆角、阴影、颜色
4. **响应式**：移动端、平板、桌面端都要完美适配
5. **可访问性**：足够的对比度、清晰的层级

## 7. 响应式断点

```
移动端 (Mobile):
- 宽度: < 640px
- 布局: 单列
- 字号: 缩小 10-20%
- 间距: 缩小 20-30%

平板 (Tablet):
- 宽度: 640px - 1024px
- 布局: 2列
- 字号: 正常
- 间距: 正常

桌面端 (Desktop):
- 宽度: > 1024px
- 布局: 3-4列
- 字号: 正常
- 间距: 正常

超大屏 (Extra Large):
- 宽度: > 1536px
- 布局: 最大宽度限制 1200px，居中
```

## 13. 可访问性要求

```
对比度：
- 正文文字与背景对比度 ≥ 4.5:1
- 大标题与背景对比度 ≥ 3:1

交互：
- 所有可点击元素有明显悬停状态
- 焦点状态有清晰的外发光或边框
- 按钮最小尺寸 44x44px（移动端）

颜色：
- 不仅依赖颜色传达信息（配合图标、文字）
- 错误状态：红色 + 错误图标 + 错误文字
```
